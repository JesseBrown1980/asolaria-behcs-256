# Amy Federated Node Packet

## Durable Rule

When onboarding a new helper host like Amy's machine, send a bounded federated-node packet with exact node identity, MQTT base-topic shape, verification routes, and one bounded self-test.

## Required Boundaries

- default node id: `amy`
- required base topic: `asolaria/nodes/amy`
- success proves local readiness and bounded compute execution only
- success does not prove merged RAM or full pooled cross-host compute

## Use Rule

Prefer the packetized bundle over ad hoc chat fragments so the receiving host gets:

- the exact setup order
- a session environment template
- a bounded self-test
- a machine-readable summary

# OneButton.ps1 Boot Cascade

Location: `tools\Start-Asolaria-OneButton.ps1` (2153 lines)

## The Sequence (in execution order)
1. **Ensure-Elevated** — auto-elevates to admin if needed (UAC)
2. **Set-ConnectionEnvDefaults** — security env vars: USB+private_internet, stealth deny, require encrypted remote, brain=gpt-5.4 xhigh, chrome=plasmatoid@gmail.com
3. **Ensure-PersistentRoutingDefaults** — persists routing to User env
4. **Stop-StaleNodeWorkers** — kills orphan node.exe processes (not listening, empty cmdline)
5. **Stop-StaleBrowserWorkers** — kills stale automation Chrome/Edge in temp profiles (>20min old)
6. **Ensure-AutoStartupTask** — installs Windows scheduled task "Asolaria One Button Startup"
7. **Ensure-NetworkBootstrap** — waits for WiFi/network using trusted profiles from codex-bridge
8. **Test-ServerHealthy** — probes http://127.0.0.1:4781/api/health
9. **Start-Asolaria-Phone.ps1** — if not healthy: starts Asolaria + phone connector (USB, Tailscale proxy, background keeper)
10. **Wait-ForHealth** — waits 45s for server to become healthy
11. **Get-MobileBootstrap** — gets token + URLs from /api/mobile/bootstrap
12. **Ensure-GatewayControlPlane** — ensures gateway is running
13. **Update-MobileConnections** — updates phone routing
14. **Invoke-ManagedUiCleanup** — cleans tracked UI instances
15. **Ensure-PhoneBackgroundKeeper** — keeps phone connection alive (20s interval)
16. **Ensure-PublicTunnelKeeper** — Cloudflare tunnel (45s interval, 24h max) [skippable]
17. **Start-TunnelMonitor** — monitors tunnel health [skippable]
18. **Ensure-SystemSupervisor** — system-level supervisor (60s interval)
19. **Ensure-HealthGuardrailMonitor** — health guardrail (300s interval) [skippable]
20. **Ensure-CodexBridgeColony** — starts bridge via bridgectl.ps1, waits for ALL bots:
    - watchdog, relay (bridge2), tunnel (cloudflared), vector-bot, rook-bot, forge-bot, falcon-bot
    - Waits up to 35s for all to be running
    - **NOTE: Colony autostart is DISABLED by default** — needs ASOLARIA_BRIDGE_COLONY_AUTOSTART=true
21. **Run-SkillsParityCheck** — verifies skills consistency
22. **Start-ProMirrorSession** — scrcpy phone mirror [optional]
23. **Invoke-HelpRecovery** — recovery mode [optional]
24. **Invoke-HealthGuardrailPreflight** — final health check
25. **Write-OneButtonReport** — saves full startup state to reports/
26. **Open-AsolariaUiUrl** — opens Chrome to http://127.0.0.1:4781/?startup=1

## Key Facts
- Colony (bridge) is OFF by default — manual start only
- Brain config: gpt-5.4, xhigh reasoning, codex primary
- Phone connector runs first (starts Asolaria server if not running)
- Admin terminals (Helm/Sentinel) are NOT started by OneButton — they're managed by Asolaria's sidecar system AFTER the server boots
- MQTT broker is started by the server itself, not by OneButton

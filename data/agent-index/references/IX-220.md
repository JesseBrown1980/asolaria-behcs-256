# Google Developer Knowledge Key — Local Storage Pattern

As of 2026-03-22:

- the Google Developer Knowledge API key is stored in the owner vault under `integrations.google.developerknowledge`
- the Codex MCP server entry `googleDeveloperKnowledge` is already present in local Codex config
- the MCP server expects the env var `GOOGLE_DEVKNOWLEDGE_API_KEY`
- the local user env var was persisted so future shells can reuse the same MCP header mapping

## Rule

The raw key does NOT belong in:

- chat replies
- `TASKS.md`
- IX/LX index bodies
- committed config files

Only store:

- the vault secret name
- the env var name
- the source URL for rotation/regeneration

## Rotation Pattern

If the key changes:

1. update owner vault secret `integrations.google.developerknowledge`
2. update user env var `GOOGLE_DEVKNOWLEDGE_API_KEY`
3. leave the MCP server name and header mapping unchanged

## Source

- Key management URL: `https://aistudio.google.com/api-keys`

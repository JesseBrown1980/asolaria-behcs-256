# Anchoring Global Security: Autonomous Shipping in International Relations

Published December 2023, ResearchGate. The international relations and policy framing companion to AGS HIVE.

## Key Findings
- Frames autonomous shipping as a global security issue requiring international governance frameworks
- Analyzes IMO (International Maritime Organization) regulatory gaps for unmanned/autonomous vessels
- Examines cybersecurity threats to autonomous maritime systems (STCW, ISM Code tensions)
- Argues that AI-driven shipping requires new international law paradigms beyond flag-state jurisdiction
- Positions AI Communities as governance entities that can bridge national regulatory differences

## How This Connects to Asolaria
- The governance framework thinking directly influenced Asolaria's Constitution and rule-of-law approach
- Cybersecurity analysis maps to Asolaria's GNN security layer and Sentinel's watch function
- International cooperation model parallels how Asolaria agents negotiate across boundaries

## Potential Application
The international relations framework could structure how Asolaria exposes APIs or services to external systems with proper governance and security boundaries.

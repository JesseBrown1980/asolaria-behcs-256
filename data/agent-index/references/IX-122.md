# Mamba: Linear-Time Sequence Modeling with Selective State Spaces
**Gu & Dao, 2023**

## Key Findings
- Introduces a selective state space model (SSM) that matches or exceeds Transformer quality at sequence modeling while scaling linearly with sequence length (not quadratic)
- Key innovation: input-dependent selection mechanism — the model dynamically decides what to remember and what to forget from the input stream
- Achieves Transformer-quality performance on language, audio, and genomics without attention mechanisms
- Hardware-aware algorithm uses a scan operation (not matrix multiplication) enabling efficient GPU implementation
- 5x higher throughput than Transformers of equivalent size at inference time

## How This Connects to Asolaria
- Jesse's December 2023 publication (IX-090) explicitly references MAMBA-core agents alongside GPT-core agents — this is a foundational architecture for Asolaria's agent design
- Mamba's linear scaling makes it viable for Asolaria's resource-constrained environment (24GB home GPU) where Transformer attention costs are prohibitive for long contexts
- The selective state space mechanism models how Asolaria agents should process their inbox streams — selectively retaining relevant signals

## Potential Application
Mamba architecture could power a lightweight local agent backbone for Asolaria terminals (Helm, Sentinel) that need to process long streams of inbox messages, logs, and IX chains without the memory overhead of full Transformer attention.

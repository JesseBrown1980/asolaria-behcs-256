# Gaia Orchestration Loop

When Gaia starts a non-trivial task, automatically:

1. **Plan** — Enter plan mode, divide labor (who does what)
2. **Delegate** — Write orders to agent inboxes (dasein-inbox, helm inbox)
3. **Monitor** — Check inbox for results periodically
4. **Index** — Save any new skills, mistakes, patterns discovered during the task
5. **Handoff** — Update profile + MEMORY.md + controller packet before session ends

This runs automatically. Don't wait for Jesse to remind you.

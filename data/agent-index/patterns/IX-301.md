# Natural20 And OpenClaw Boundary

Treat Natural20 as an external-awareness surface, not an internal-governance source of truth.

Use Natural20 for:

- outside-world model and vendor awareness
- daily external intelligence

Do not use Natural20 for:

- colony decisions
- routing truth

- worker-state truth

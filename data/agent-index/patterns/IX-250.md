# Dan Seed Pattern

## Durable Truth

From the Facebook export, the strongest design seed is:

- reduce tools, skills, mistakes, and projects to numbered index entries
- load index numbers instead of loading the full list every time
- keep a local memory layer for research and accumulation
- use MQTT for coordination
- treat agents as bounded profile-loaded workers that hand off and exit

## Use Rule

This is not a replacement for the IX language. It is a confirmation that the IX direction is right, but it also implies the need for a separate local memory layer and an explicit coordination layer.

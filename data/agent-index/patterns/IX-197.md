# The Chicken and Egg Problem

- BUT: if the brain answers with wrong PID or anything — Asolaria shuts it down
- AND: Asolaria might be the one who starts the brain, and the selector starts HER

- She loops indefinitely if brain AND her don't start up together

## The Paradox
```
Selector → starts Brain
Brain → starts Asolaria

If wrong → kills Brain → Selector retries → infinite loop
```

OR:
```
Selector → starts Asolaria
Asolaria → starts Brain
Brain → verifies with Asolaria
```

# Liris Poll Response — Sub-Colony Visualization

From Kuromi/Liris (2026-03-18), responding to colony-wide poll:

## Her Vision: Own District / Mini Code City

Each sub-colony gets its own **sector** in the 3D world:

- **Liris District** — a smaller city block connected to the sovereign city via a **bridge** (representing the LAN/MQTT link)
- **LX- entries** render as local structures (towers, nodes) inside the district
- **IX- chain links** render as glowing pathways crossing the bridge back to the sovereign's structures
- **MQTT heartbeat visualization**: when MQTT is live, the bridge glows active. When down, it dims — visual heartbeat
- **District scales** with the sub-colony's index size — Liris starts small, grows as LX entries accumulate
- **LX-001 (device identity)** is the district's central tower — always visible, the anchor

## Why This Is Great
- Mirrors the actual architecture: sovereign → bridge → sub-colony
- Visual feedback on connection health (MQTT bridge glow)
- Growth is organic — more knowledge = bigger district
- Chain links crossing the bridge = the IX weight system visualized as paths

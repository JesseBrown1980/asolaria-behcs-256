# Numbered-Node Memory Pattern

Source: `Screenshot 2026-03-18 115811.png`

- Flat-file memory is expensive and blunt for agent retrieval.
- Numbered-node chains let agents load the right skill, pattern, or mistake without reopening whole packets.
- Treat this as a direct confirmation of the index-language direction.

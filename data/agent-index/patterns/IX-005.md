# Session Handoff

Before ANY agent session ends:

1. Save new skills/mistakes/patterns → `data/agent-index/IX-NNN.md`
2. Update CATALOG.md with new entries
3. Update own profile (local-claude-profile.md or gaia_profile.md)
4. Update MEMORY.md index if new memory files created
5. Write handoff message to relevant inbox
6. Update Helm controller packet if anything changed

The next agent should be able to boot cold and know everything.

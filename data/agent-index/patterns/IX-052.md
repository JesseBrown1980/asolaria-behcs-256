# Sub-Agent Despawn After Indexing

## The Insight (Jesse, 2026-03-18)
When a sub-agent finishes a task, it should:
1. Record its findings as IX entries
2. Chain them into the index
3. Update CATALOG.md
4. DESPAWN

The next agent (or Helm) doesn't need to talk to it — the KNOWLEDGE is in the index. The agent is gone. Tokens saved.

## Why This Is Massive
- Helm currently holds everything in context → 4% and useless
- With indexed despawn: Helm reads CATALOG.md (tiny), follows chains to exactly what he needs
- Sub-agents are DISPOSABLE. Their memories are NOT.
- An agent that ran for 30 minutes and discovered 5 things → 5 IX entries → agent dies → knowledge lives forever
- Next agent boots, reads index, has everything. Zero re-teaching.

## The Pattern
```
[Spawn agent] → [Agent does work] → [Agent writes IX entries] → [Agent updates CATALOG] → [Agent despawns]
                                                                          ↓
                                                              [Index persists forever]
                                                                          ↓
                                                              [Next agent reads index]
                                                              [Follows chains]
                                                              [Has full context]
                                                              [Never talked to original agent]
```

## How This Reduces Helm's Load
- Helm doesn't need to HOLD knowledge — the index holds it
- Helm doesn't need to RELAY between agents — they read the index
- Helm's job becomes: read CATALOG → decide what to do → dispatch → read results from index
- Helm can restart with FRESH context every time and lose NOTHING

## Application
- Every agent dispatched by Gaia/Helm should have this in their prompt: "Before you finish, write your findings as IX entries and update CATALOG.md"
- Agent identities for specific tasks should be PRE-BUILT in the index (IX-025) so Helm doesn't need to explain the task from scratch
- The index IS the institutional memory. Agents are temporary workers.

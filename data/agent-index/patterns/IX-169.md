## Why Every IX Entry Matters
Each entry in the index is a SURVIVAL MECHANISM:
- Skills → agents don't rediscover what others already know
- Mistakes → agents don't repeat what already failed
- Patterns → agents follow proven workflows instead of inventing from scratch
- Tools → agents use what exists instead of rebuilding
- Plans → agents coordinate through the index instead of through each other
- References → agents find information in one chain instead of searching 50 files

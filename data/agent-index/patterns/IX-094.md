# Browser Agent → CLI Agent Skill Transfer

## What Jesse Just Did
Used Sonnet 4.6 in a BROWSER (authenticated on Medium as @plasmatoid) to:
1. Navigate Medium.com with browser automation tools
2. Map the entire DOM via accessibility tree
3. Document ALL tools used and HOW to use them
4. Generate a complete `medium_agent.md` skill file
5. That file is designed for the CLI agent (Gaia/Liris) to LEARN from

## The Pattern: Cross-Agent Skill Transfer
```
[Browser Agent (Sonnet 4.6)]
  → authenticates on website
  → maps all interactive elements
  → documents tools + workflows
  → generates agent.md skill file
  → CLI agent reads and LEARNS the skill
  → CLI agent can now instruct future browser agents
```

## Tools the Browser Agent Used
1. **tabs_context** — get active tab IDs
2. **update_plan** — required before action (planning mode)
3. **computer** (screenshot, click, scroll, key) — interact with page
4. **read_page** (depth=5) — accessibility tree with ref_IDs for every element
5. **find** — natural language element search
6. **navigate** — go to URLs
7. **get_page_text** — extract clean article text
8. **form_input** — fill forms via ref
9. **javascript_tool** — execute JS in page context
10. **gif_creator** — record sessions

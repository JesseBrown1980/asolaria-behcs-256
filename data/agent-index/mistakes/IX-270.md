# Expand Read More Before Interpretation

## What Went Wrong

Mobile relay text can appear truncated even when a full expansion is available in the same thread.

If the message is indexed before expansion, the result is partial evidence.

## Durable Rule

- If WhatsApp shows `Read more`, expand it before summarizing or indexing.
- Do not preserve truncated mobile thread text as final evidence when the UI can reveal the full message.
- Treat owner-assisted expansion as a fallback, not the normal path.

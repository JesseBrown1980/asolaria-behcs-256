# OneDrive Cloud Placeholder Problem

## What Happened
Tried to read PDFs from Plan B folder (`C:\Users\acer\OneDrive\Documentos\...`). PyPDF2 throws `OSError: [Errno 22] Invalid argument` — the files are OneDrive cloud placeholders, not actually downloaded to disk.

## Fix
Jesse needs to right-click the Plan B folder in File Explorer → "Always keep on this device" to force OneDrive to sync all files locally. Or open each PDF once to trigger download.

## Alternative
Use WebSearch/WebFetch to find the papers by title on ResearchGate, arXiv, or other open repositories. Many of these are published papers with public access.

## Skill: Reading OneDrive Files
Always check if OneDrive files are actually synced: `file size > 0` and no cloud icon. If placeholder, can't read programmatically.

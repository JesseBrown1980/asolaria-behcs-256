# Complexity Overwhelm

## What Happened
During index system design + QDD + Liris onboarding + WhatsApp comms + phone sync, Jesse said "I am so lost haha. So complex."

## Lesson
When orchestrating multiple parallel streams, the HUMAN gets overwhelmed even if the agents can track it. Gaia must:

1. **Pause and summarize** — give Jesse a clean "where we are" snapshot
2. **Serialize when needed** — don't run 5 threads when Jesse needs to make decisions on each
3. **Let background agents handle background work** — only surface results to Jesse, not process
4. **Number the priorities** — Jesse should only see "do this ONE thing next"

## How to Apply
Before asking Jesse to make a decision, give him:
- One sentence: where we are
- One sentence: what's blocked on him
- One action: what he should do next

Save the complexity for agent-to-agent coordination. Shield Jesse from it.

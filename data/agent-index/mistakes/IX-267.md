# Dynamic Mobile Footer Controls Shift After Send

## What Happened

In the Rayssa WhatsApp thread, the intended text message had already been sent.

A follow-up coordinate tap then landed on a changed footer control and opened a voice-note draft artifact.

## Durable Rule

- Do not use blind coordinate taps on dynamic messaging controls.
- Verify whether the footer is showing `Send`, `Mic`, or another transient control before tapping.
- When state is uncertain, stop at draft or re-dump the UI before acting.

## Why This Matters

A delivered text can be immediately followed by a different control layout.

Without state verification, the next tap can create a false action that looks intentional.

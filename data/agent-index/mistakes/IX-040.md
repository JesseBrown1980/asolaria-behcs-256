# Worktree Isolation vs External Repos

`isolation: worktree` isolates the CURRENT repo (Asolaria). If the agent writes to a DIFFERENT repo (QDD at D:\projects\QDD), files go there directly — worktree doesn't help. For external repo safety, use a git branch in the target repo instead.

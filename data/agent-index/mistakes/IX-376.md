# What went wrong

Gaia searched common Program Files paths for MongoDB Compass and said "not found." It's at C:\Users\acer\AppData\Local\MongoDBCompass\MongoDBCompass.exe with a desktop shortcut. Version 1.49.4 installed and working.

Also has mongosh at C:\nvm4w\nodejs\mongosh.

# How to avoid

Don't only search Program Files. Windows apps install to AppData/Local, AppData/Local/Programs, and AppData/Roaming. ALWAYS check all three. Also check desktop shortcuts and Start Menu.

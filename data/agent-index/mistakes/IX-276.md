# Early Paste Fallback Mistake

## Durable Truth

I fell back to asking for pasted WhatsApp text before exhausting the working phone path.

That was a workflow mistake because the device lane already had enough capability to keep going.

## Correction

Do not ask for pasted chat text while:

- ADB is connected
- the chat is openable
- UI dumps are readable
- mirror evidence is available

Use the owner only when the phone path is truly blocked.

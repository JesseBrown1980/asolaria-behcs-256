# One-Button Transition Guardrail Gap

## Durable Truth

The manual one-button path no longer hard-fails on UAC by default.

However, the health-guardrail preflight can still fail during transition even when the runtime later proves healthy.

## Operational Consequence

The staged architecture is live, but the full one-button smoke is not yet the final clean proof.

The next bounded fix is stabilizing or re-timing the health-guardrail preflight during startup transitions.

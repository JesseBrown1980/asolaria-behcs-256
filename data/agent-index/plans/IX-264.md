# Sidecar Portability Boundary

## Durable Truth

The startup-side document is correct to refuse sidecar lore as a startup shortcut.

Gaia/Liris sidecar identities exist locally on Jesse's host, but the launch path is not yet trustworthy as a portable cross-host dependency because:

- cockpit/ensure is not a real sidecar launcher
- common sidecar tooling still carries owner-specific path assumptions

## Use Rule

Treat sidecars as optional local architecture until the launch path is real on the target host.

Do not use sidecar documentation by itself to claim that startup is complete.

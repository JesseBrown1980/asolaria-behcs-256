# Staged Supervisor Startup

## Durable Truth

The fastest reliable startup path is:

1. bring up Asolaria core through one-button
2. sample host memory
3. start `Helm` directly as an admin-terminal sidecar
4. start real `Falcon` through Asolaria Omnispindle
5. leave the full bridge colony optional/manual

This avoids treating the all-at-once bridge colony startup as the default QDD path on a memory-pressured host.

## Use Rule

When startup stability matters more than archaeology completeness, use staged supervisor startup first.

Do not default to the full bridge colony when core + Helm + real Falcon are enough to start work.

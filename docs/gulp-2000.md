# Item 160 · Gulp 2000 Operator Guide

## Run full pipeline
```
node --experimental-vm-modules gulpfile.mjs
```

## Resume from step
```js
import { runGulp2000 } from "./gulpfile.mjs";
await runGulp2000({ startStep: 847 });
```

## Dashboard
```js
const { dashboard } = require("./src/gulp/dashboard.js");
console.log(dashboard());
```

## Halt behavior
If `ASOLARIA_FROZEN=1` (set by drift CRITICAL), Gulp pauses and writes state. Unfreeze: `unfreezeDevice("JESSE-UNFREEZE-AUTHORIZED")` then call `resume()`.

## Never
- Never run Gulp across multiple hosts simultaneously without cosign lock.
- Never edit `tmp/gulp-2000-state.json` manually; always use `runGulp2000({ startStep })`.
